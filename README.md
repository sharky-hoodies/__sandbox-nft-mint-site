## Setup
1. .env.localを更新
- NEXT_PUBLIC_CONTRACT_ADDRESS: NFTコレクションのコントラクトアドレス
2. ローカル起動
```
yarn install && yarn dev
```
Open http://localhost:3000 with your browser to see the result.
